# 🚌 BusGo

## Descripción

BusGo es un Producto Mínimo Viable (MVP) desarrollado con el objetivo de facilitar la gestión y consulta de rutas de transporte público mediante una plataforma web.

El sistema permite administrar información relacionada con rutas, recorridos y usuarios, ofreciendo una solución sencilla para mejorar el acceso a la información del transporte.

## Objetivos

* Gestionar rutas de transporte.
* Consultar recorridos disponibles.
* Administrar usuarios del sistema.
* Centralizar la información en una base de datos.

## Tecnologías Utilizadas

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* XAMPP

## Funcionalidades

* Inicio de sesión.
* Registro y gestión de usuarios.
* Crear rutas.
* Editar rutas.
* Eliminar rutas.
* Visualizar recorridos.
* Conexión con base de datos MySQL.

## Instalación

1. Clonar el repositorio.
2. Copiar el proyecto en la carpeta `htdocs` de XAMPP.
3. Crear la base de datos `busgo`.
4. Importar el archivo SQL del proyecto.
5. Iniciar Apache y MySQL.
6. Acceder desde:

```
http://localhost/BusGo/public
```
